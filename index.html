Política de privacidad de Doorio

Última actualización: 23 de junio de 2026

Doorio es una aplicación personal que abre puertas y portales enviando peticiones HTTP a los endpoints que tú mismo configuras. Esta política explica qué datos maneja la app y cómo.

Resumen

Doorio no recopila, transmite ni comparte tus datos personales con el desarrollador ni con terceros. No hay analítica, ni publicidad, ni servidores propios. Todo lo que configuras se guarda únicamente en tu dispositivo.

Datos que se almacenan en el dispositivo

  • Configuración de tus entradas: nombre, color, método HTTP, URL del endpoint, cabeceras y cuerpo opcional. Se guardan en una base de datos local del dispositivo.
  • Tokens de autenticación (opcionales): si una entrada usa un token Bearer, se cifra en reposo mediante el Android Keystore y no abandona el dispositivo.
  • Historial de aperturas: las últimas aperturas de cada entrada, también en local.

Estos datos no se envían a ningún servidor controlado por el desarrollador y se eliminan al desinstalar la app.

Ubicación

Si activas la apertura automática (geofencing), Doorio usa tu ubicación únicamente en el propio dispositivo para detectar cuándo llegas a un punto que has definido y abrir esa puerta. La ubicación no se almacena en remoto ni se transmite a ningún tercero; la gestiona el sistema operativo para activar el geofence. El permiso de ubicación en segundo plano se usa solo para que la apertura automática funcione con la app cerrada.

Conexiones de red

  • Tus endpoints: al abrir una entrada, la app realiza la petición HTTP exclusivamente a la URL que tú has configurado.
  • Mapa: al elegir un punto en el mapa, se descargan mosaicos de OpenStreetMap desde sus servidores, sujeto a la política de privacidad de OpenStreetMap.

Permisos

  • INTERNET / ACCESS_NETWORK_STATE: enviar las peticiones de apertura y cargar el mapa.
  • ACCESS_FINE/COARSE/BACKGROUND_LOCATION: apertura automática por geofencing.
  • POST_NOTIFICATIONS: avisar cuando una puerta se abre automáticamente.
  • RECEIVE_BOOT_COMPLETED: volver a registrar los geofences tras reiniciar el dispositivo.

Menores

Doorio no está dirigida a menores y no recopila datos de ninguna persona.

Cambios

Si esta política cambia, se actualizará la fecha de arriba.
